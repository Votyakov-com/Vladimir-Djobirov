1) Класс - некое вместилище данных, которые представляют собой несколько объектов. Например у нас есть класс Автомобиль. Данный класс может содержать такие
переменные как: марка, модель, год выпуска, а также методы: "завести двигатель", "пристегнуть ремень", "поехать". То есть по сути класс - это какой-то шаблон/слепок.

2) Параметры класса - это такие переменные, которые в общем характеризируют и сам класс. К примеру у собаки есть такие параметры как: возраст, порода,
любимая еда, имя и т.д.

3) Функции класса - это некие методы класса , которые представляют собой какие-то операции, которые выполняются над объектами данного класса. 
Например, у нас есть класс: Банковский счёт. Допустим у данного класса есть объект: баланс, который показывает сумму на счёте. К данному объекту может быть
применена функция: Снять деньги, которая уменьшает баланс на какое-то значение t.
